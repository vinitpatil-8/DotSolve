# DotSolve AI

## Project Description
DotSolve is a web application developed using React, TypeScript, Tailwindcss and Vite. This application allows users to solve connect-the-dots puzzles interactively, providing an engaging experience by blending learning with fun.

## Features
- Interactive puzzle solving
- Responsive design for various devices
- User-friendly interface
- Easy installation and setup
- Contribution guidelines for community involvement

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vinitpatil-8/DotSolve.git
   ```
2. Navigate into the project directory:
   ```bash
   cd DotSolve
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage Guide
Once the development server is running, open your browser and go to `http://localhost:3000` (or the port specified in your terminal). You can then start solving connect-the-dots puzzles.
